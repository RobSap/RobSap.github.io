Tide Prediction

Description

Each of these projects are expanded off each other. Noted are the key differences.

Tide prediction Version 1: one location, all displayed on 1 page. Uses sqlite database to store requests for future use, use soap request to get tide infromation form NOAA.

Tide prediction Version 2: three locations, display specific dates, cleaned up display, check sqlite before new soap request.

Tide prediction Version 3: select date specific date range, Included most Oregon Beaches, displays 2 days by default, user can select up to 5, Uses sqlite database to store 5 days, now pulls in 2 days before request and 2 days after request.

Project Code: Version 1 -- Version 2 -- Version 3.
