Unity Based Augmented Reality Applications

These projects were created using Vuforia and Unity.

The focus of this project was to use a background image and map a 3d animation to that image. Vuforia takes care of mapping a the background image, and Unity us used for displaying the animation. Then using a phone, you can see the different mappings and character animations.
